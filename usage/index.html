disclaimer / 免責事項

this app has function access to Imgur account, but don't have function to create new Imgur account. please visit register page on Imgur website to create new account.

this app don't have any function to browse Imgur's gallery and albums. this app's upload history just show data sent from this app and installed machine.

this app can not help about server-side troubles.

このアプリにはImgurアカウントと連動する機能がありますが、 Imgurアカウントを新しく作成する機能はありません。 アカウントの作成はImgurのWebサイトで行なってください。

このアプリにはImgurのデータを閲覧する機能はありません。 アップロード履歴に表示されるのは、このアプリから送信したデータだけです。

Imgurのサーバの不調による問題はこのアプリでは解決できません。

basic usage / 単純な使い方

when open this app from launcher, image is not selected at initial state.

please push pick button to choose image.

このアプリを起動すると、最初は画像が選択されていません。

選択ボタンを押して画像を選択しましょう。

list of image picker app will be shown.

if you want to skip choosing app at next time, please enable "Use by default for this action" checkbox.

画像を選択するアプリの一覧が表示されます。

もし次回からアプリの選択をスキップしたい場合、「常にこの操作で使用」チェックボックスを有効にしてください。

in this example, using "QuickPic" to pick up image.

この例ではQuickPicを使って、画像を選択します。

oops, picked image was tilted. please push edit button to fix it.

if you check "auto start editor" in application settings, editor screen is automatically open after pick or capture image.

おや、画像が傾いてますね。editボタンを押して修正しましょう。

設定で「画像加工を自動開始」を有効にすると、画像選択やカメラ撮影の直後に自動的に編集画面を開きます。

this app has in-app editor to rotate,resize,crop image.

on push save button, new image will be created.

there are auto-rotate and JPEG quality option in application settings.

editor では 画像の回転、リサイズ、切り抜きを行うことができます。

saveボタンを押すとその場で新しい画像が作成されます。

自動回転やJPEG品質のオプションは設定画面にあります。

second line from bottom in the screen is used to choose upload account and album.

but, Imgur allows anonymous upload without account.

in this example, choose "(anonymous)".

if you want to upload with Imgur account, see section account authentication and albums.

画面の下から２行目で、アップロードするアカウントやアルバムを選択できます。

が、アカウントを設定しなくても画像のアップロードはできます。

今回の例では (anonymous) / (名無し) を選択してアップロードします。

もしImgurアカウントを使ったアップロードを行いたい場合、アカウント認証やアルバムの項を参照してください。

on push upload button, uploading will be start.

on complete upload, URL of uploaded image will be copied to clipboard. you can paste URL to any text input on other app.

アップロードボタンを押すとアップロードが行われます。

アップロードが完了すると、URLがクリップボードにコピーされます。他のアプリのテキスト入力部で貼り付け操作を行うことで、URLをそこに入力できます。

work as IME extension / IME拡張としての動作

this app support Mushroom IME(Input Method Editor) extension. when start this app from IME, this app work as IME extension and automatically input URL of uploaded image.

this function depends on IME. if your IME has no support of Mushroom IME extension, please skip this section. also, how to start IME extension depends on what IME you use. in this example, ATOK is used.

このアプリはMushroom IME拡張に準拠しています。Mushroom をサポートしているIMEからImgurマッシュを起動することで、画像の選択→アップロード→URLの入力をシームレスに行うことができます。

マッシュルームの起動方法はお使いのIMEによって異なります。この例では ATOK での起動方法を説明します。

open some app that has text input.

then, long-press left-bottom icon on softwea keyboard of ATOK.

テキストエディタやTwitterアプリなど、テキストを入力できるアプリをまず開きます。

次に、ATOKのソフトウェアキーボードの左下のアイコンを長押しします。

slide your finger to abobe mushroom icon.

"ATOKダイレクト" will be shown.

this screen shot shows icon of this app to right of mushroom icon, but it will not be shown at first time, because it is list of last recently used.

untouch your finger that on mushroom icon.

そのまま指を上の、キノコのマークにスライドします。

ATOKダイレクトが表示されます。

この画面写真ではキノコのマークの右にImgurマッシュのアイコンが出ていますが、これは起動履歴なので最初は見つからないでしょう。

キノコのマークをタッチしている指を、画面から離します。

list of installed mushroom extension will be shown.

choose Imgur Mushroom.

端末にインストールされたマッシュルーム拡張の一覧が表示されます。

Imgurマッシュを選びます。

following instruction is similer to basic usage. choose image, upload it.

Imgurマッシュの画面が開いて、画像選択からアップロードまでの流れは単純な使い方の説明と同じです。

on complete upload, URL will be automatically input via IME.

アップロードが終わるとURLが自動的にIME経由で入力されます。

upload history / アップロード履歴

this app has in-app upload history. image uploaded as anonymous can be deleted only from upload history.

このアプリは内部にアップロード履歴を保存します。名無しでアップロードした画像は、アップロード履歴からのみ削除可能です。

open this app, then push menu key , then choose history.

このアプリを起動して、次にmenuボタンを押して、次に履歴を選択します。

list of the uploaded image from this app will be shown.

アプリからアップロードした画像の一覧が表示されます。

on push list item, context menu for check/delete upload history will be shown.

リストから項目を選択すると、履歴を確認/削除するメニューが表示されます。

Input this URL
URL of this image will be copied to clipboard. or if you start this app as IME extension, URL will be input via IME.
Browse image URL
URL of this image will be open via web browser.
Browse Imgur's site
URL of image information page will be open via web browser.
delete image on the server
uploaded image will be delete from on Imgur's server.
delete history
delete only history data in this app. uploaded image on Imgur's server will not be deleted.
このURLを入力
アップロードされた画像のURLを、クリップボードにコピーします。 ただし、このアプリをIME拡張として起動した場合はURLは直接IMEに入力されます。
画像リンクを開く
アップロードされた画像のURLを、Webブラウザで開きます。
画像詳細ページ
アップロードされた画像の詳細ページを、Webブラウザで開きます。
サーバから削除
アップロードされた画像を、Imgurのサーバから削除します。
履歴だけ削除
アップロードされた画像を、このアプリのアップロード履歴から削除します。Imgurのサーバからは削除されないので注意して下さい。

account authentication / アカウント認証

this app has function access to Imgur account, but don't have function to create new Imgur account. please visit register page on Imgur website to create new account.

このアプリにはImgurアカウントと連動する機能がありますが、 Imgurアカウントを新しく作成する機能はありません。 アカウントの作成はImgurのWebサイトで行なってください。

open this app ,then push menu key, then choose settings.

このアプリを開いて、次にメニューボタンを押して、次に設定を選びます。

choose "account authentication" from list of settings.

設定のリストから「アカウント認証」を選択します。

please input information to be required for login.

認証に必要な情報を入力します。

push "allow" button at righit-bottom edge in login form.

ログイン画面の右下の「Allow」ボタンを押します。

authentication will be complete in a few minutes.

すこし待つと、認証が完了します。

choose account after back to upload screen.

アップロード用の画面に戻って、登録したアカウントを選択します。

albums / アルバム

on complete account authentication, you can choose albums that created at Imgur's web site.

or create new album by using rightside "+" button.

アカウント認証が終わると、ImgurのWebサイトで作成したアルバムを選択することができます。

また、右端の ＋ ボタンを押して新しくアルバムを作成することもできます。

please input title and other information when create new album.

alphabet,number, space, hyphen, underline character will be allowed as letter in album title.

push OK to create new album.

アルバムを作成する時は、アルバムのタイトルなどを入力します。

アルバム名に指定できる文字は半角アルファベット、半角数字、半角スペース、半角ハイフン、半角アンダーラインだけです。

OKボタンを押すとアルバムが作成されます。

if you want to delete album, long press the box of album selection.

アルバムを削除したい場合、アルバムを選択する部分を長押ししてください。

if you want to delete album, press OK in the confirmation dialog.

no recovery after delete albums. be careful.

アルバム削除の確認でOKを押すと、アルバムが削除されます。

削除してしまうと元に戻せません。注意して下さい。

bulk upload / 一括アップロード

if you have the image viewer app that has capavility of multiple-selection and send share intent, this app can bulk upload the multiple image all together.

this example shows multiple selection using QuickPic and bulk upload with this app.

複数選択と共有のできる画像閲覧アプリ(QuickPicやES File Explorerなど)から共有されることで、 複数の画像を一括アップロードすることができます。

この例ではQuickPicでの複数選択と、そこからImgurマッシュで一括アップロードを行う手順を説明します。

open any folder from image viewer app such as QuickPic.

push check-boxed button at top-right edge.

some machine shows pencil button for multiple selection.

QuickPic等の画像閲覧アプリで、適当なフォルダを開きます。

右上の、チェックボックスのついたボタンを押します。

一部の端末では鉛筆マークのボタンが該当するようです。

the top-side view will be changed.

then, select some images.

画面上部の表示が変わりました。

画像をいくつか選択します。

push share button at top-center edge.

選択したら、画面上部にある共有ボタンを押します。

list of app that receive share intent will be shown.

choose "Imgur bulk upload".

複数ファイルの共有を受け取れるアプリが一覧表示されます。

「Imgur一括投稿」を選択します。

bulk upload screen of this app will be shown. oops, some images was tilted.

このアプリの一括アップロード画面が開かれます。おや、いくつか画像が傾いてますね。

tap image to edit or remove from upload list.

画像をタップすると、個別に編集やアップロード対象リストからの除去を行えます。

on push upload button, bulk upload will be start.

on complete upload, URL of images will be copied to clipboard.

アップロードボタンを押すと、まとめてアップロードが行われます。

アップロードが完了するとURL(複数)がまとめてクリップボードにコピーされます。
